--INSTRUÇÕES--

 - cada tecla tem o formato:   
	          b
                c a d
	          e

- para escolher a letra 'a', basta um clique
- para escolher 'b', deslizar o dedo para cima (carregando na tecla)
- para escolher 'c', deslizar o dedo para a esquerda
- etc...

ESCOLHER UMA PALAVRA RECOMENDADA:
 - na secção não interativa, irão aparecer 3 recomendações de palavras.
 - para selecionar uma, clicar no botão com 3 setas e deslizar com o dedo na direcção da palavra que se pretende escolher
